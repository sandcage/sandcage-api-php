<a name="requirements" />
##Requirements

* A SandCage account, in order to get your SandCage API Key. Once logged into SandCage get your API Key at https://www.sandcage.com/panel/api_key
* 5.3.10+
* Curl
* JSON


<a name="installation" />
##Installation

```php
// Include the SandCage class
include_once('SandCage.php');
```
